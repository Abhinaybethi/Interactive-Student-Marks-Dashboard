# Student Marks Dashboard (HTML5 + CSS3 + JavaScript)

A simple interactive web application to calculate student marks, percentages, and grades using pure HTML, CSS, and JavaScript.

## Features
- Input fields for 4 subjects
- Total marks calculation
- Percentage calculation
- Grade evaluation
- Clean UI with responsive design
- Input validation using JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
Open `index.html` in any browser.
